Quizzy – Smart Quiz Platform

Student Name: Prachi Gaur
Batch: 25–26
Mini Project Submission

Description:
Quizzy is a web-based smart quiz application that allows users to
log in or sign up, choose quiz categories and difficulty levels,
attempt timed questions generated dynamically using the Open Trivia API,
and view scores on a leaderboard.

Technologies Used:
- HTML
- CSS
- JavaScript
- Open Trivia Database API
- Browser Local Storage
